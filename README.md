## Setup

1. Install Docker (or another container runtime)
2. Clone repository
3. Open in devcontainer `Ctrl + Shift + P` > `open in container`
4. Load workbench `nix develop`

## Create SSH Key

```sh
ssh-keygen -t ed25519 -C "" -P "" -f ssh-id
```
